# FastApi101


Clonar projeto

```bash

> git clone <link-do-repositorio>

```

Iniciar shell poetry

```bash

> cd FastApi101

> poetry shell

```

Instalar dependências

```bash

> poetry install

```

Rodar comando runserver da CLI

```bash

> python -m fast

```
